Install description of YAML for Drupal - Grids Module

1. Rename your sites/all/modules/panels/plugins/layouts directory to
   sites/all/modules/panels/plugins/layouts.org or delete the entire directory.
2. Copy the YAML Grid Module layouts directory into sites/all/modules/panels/plugins/layouts.
